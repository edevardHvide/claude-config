# Snow-like Wind Particles

## Change
Replaced colored streamline particles (cyan -> yellow -> red gradient) with white/blue-tinted snow-like particles for a more realistic winter aesthetic.

## Details
- **Count:** 10K desktop / 4K mobile (was 6K / 2K)
- **Appearance:** White to light-blue tint, shorter trails, shorter lifespan
- **Motion:** Added turbulence for more natural snow-drift look
- **Rationale:** Colored streamlines looked like a weather map; white particles look like blowing snow and match the app's purpose of showing snow transport
