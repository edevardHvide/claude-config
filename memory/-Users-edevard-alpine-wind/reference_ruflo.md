"Ruflo plugin" = the `claude-flow` / `ruflo` MCP server already active in the environment. It powers `mcp__claude-flow__*` tools. It is NOT a frontend library or npm package to integrate into the React app.

**Repo:** https://github.com/ruvnet/ruflo
**Install:** `npx ruflo@latest` (legacy: `npx claude-flow@latest`)
**Use in project:** Available as MCP tools for orchestration, research, and agent coordination — not as a code dependency.
