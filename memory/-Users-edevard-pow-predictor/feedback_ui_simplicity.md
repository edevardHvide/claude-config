name: UI Simplicity Over Fancy
description: User prefers clean minimal UI — rejected glass pill label in favor of plain white text
type: feedback

Don't over-design UI elements. When user says "pretty", they mean clean and high-quality, not fancy/decorated.

**Why:** User rejected a glass-pill SVG label background ("loose the fancy bubble black background. White is fine, I just want quality letters in white"). Also softened a hard red button to rose.

**How to apply:** Default to minimal styling. Use subtle effects (shadows for readability, soft colors) over heavy treatments (backgrounds, borders, gradients). Ask before adding decorative elements.
