User maintains a second project at `/Users/edevard/alpine-wind` (GitHub: edevardHvide/pow-predictor).

Pow Predictor is a browser-based 3D snow redistribution simulator for alpine terrain using CesiumJS, React, TypeScript. It models wind-driven snow transport with Pomeroy-Gray saltation physics, Winstral Sx terrain exposure, and 2D advection. Has a simulation mode that fetches real weather data from NVE (Norwegian Water Resources and Energy Directorate).

The user has deep domain knowledge of snow physics and skiing — they provided the ranked physics improvement plan with specific references to Pomeroy, Li & Pomeroy 1997, Winstral Sx, Alpine3D. They care about physical accuracy but pragmatically accept browser-level simplifications.
