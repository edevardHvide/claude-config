# Claude Config

My Claude Code configuration — settings, skills, memories, and project guides. Synced via the `/learn` skill.

## Structure

```
settings/          Claude Code settings (settings.json, settings.local.json)
skills/            Global skills available across all projects
memory/            Per-project memory files (feedback, user profiles, project context)
```

## /learn Skill

Run `/learn` at the end of a session to capture learnings and sync everything here.
